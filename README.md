
## redis_testing

A repo that demonstrates how to use the [fakeredis](https://github.com/jamesls/fakeredis)
for testing.

## Setup

* Create a virtual environment
  
  `python3 -m venv .venv`

* Activate the virtual environment

  `source .venv/bin/activate`

* Install libraries

   `pip install -r requirements.txt`
  
* Run the tests

   `pytest`
